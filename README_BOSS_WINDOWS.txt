PJF Folio Checker (Windows)

1) Download the ZIP and unzip it
2) Open the folder "PJF-Folio-Checker"
3) Double-click "PJF-Folio-Checker.exe"
4) Follow the on-screen wizard:
   - Select your Excel file
   - Enter portal username/password
   - Wait for results

Outputs:
- Folios_results.xlsx
- Folios_missing.xlsx

If Windows shows a warning:
- Click "More info" → "Run anyway"
