# PJF Folio Checker (Multi-Tipo)

## Excel format
- Row 1: column headers are the **Tipo de Expediente** labels (e.g., “Amparo Indirecto”)
- Row 2..N: folios like 123/2025

## Run (Windows)
Double-click `run.bat`

## Run (macOS)
Double-click `run.command`

## Outputs
- Folios_results.xlsx
- Folios_missing.xlsx
